# Sales Forecasting Dashboard

## Overview
This project builds a sales forecasting solution using historical retail data.
A time-series model is used to predict future sales, and results are visualized using Power BI.

## Tools Used
- Python
- Prophet
- Pandas
- Power BI

## Project Workflow
1. Cleaned and structured historical sales data
2. Trained a time-series forecasting model
3. Generated future sales predictions
4. Visualized actual vs forecasted sales in Power BI

## Files
- `main.py` – Python script for training the forecasting model
- `dataset.csv` – Preprocessed sales data provided by the internship
- `forecast.csv` – Forecasted sales output
- `sales_forecast.pbix` – Power BI dashboard file

## Output
The Power BI dashboard shows historical sales trends and future sales forecasts to support business decision-making.
